cheat sheet

git

git init: Start a new repository

git add .: add files

git commit -m: commando dat veranderingen toevoegt aan je lokale repository.

git status: Om te zien welke bestanden nog moeten commiten

git clone [url]: Deze opdracht wordt gebruikt om een repository te verkrijgen van een bestaande URL.

git log: versiegeschiedenis van de huidige branch.

git branch: lijst van alle lokale vertakkingen in de huidige repository.

git pull: Deze opdracht haalt wijzigingen op de externe server op en voegt ze samen in uw werkdirectory.

git push: Kopieert de toestand van het lokale repository naar het remote repository.

it remote add origin: een remote is een online versie van jouw repository.

cmd

cd: Change directory

dir: geeft de inhoud van een directory

mkdir: Maak een nieuwe directory

rmdir: Remove directory

tree: show directory tree-structure

explorer .: opent verkenner
